prompTweet
==========

##기능
1. 익명
  - 개인정보 비공개
2. 인스턴트 메시지
  - 전체 최근 1000개만 저장 이후 메시지 삭제됨
3. 공개 채널, 비공개 채널로 구성
  - 옴니박스(주소창에 ptw 타이핑 후 tab키)로 자신의 현재 채널 선택
4. 컨텍스트 메뉴 제공
  - 오른 클릭으로 쉬운 포스팅
5. 푸쉬 알림
  - 하단 Toast 메뉴와 익스텐션 new 아이콘으로 신규 포스팅 알림

##기술 스택
1. Server
  1) Go (standalone)
2. Client
  1) Chrome Extention
  2) HTML5 Websocket
  3) AngularJs
